Oops! Erro de Configuração da API

A Tripinha não conseguiu se conectar à API da OpenAI para buscar destinos. Isso acontece quando a chave da API não está configurada corretamente.

Para corrigir esse problema:

1. Edite o arquivo config.js

Localize o arquivo assets/js/config.js e atualize a linha com sua chave da OpenAI:

openAI: "sk-sua-chave-da-openai-aqui",

2. Certifique-se que:

  • A chave começa com sk- (não use o formato sk-proj-)
  • Sua conta OpenAI tem saldo disponível
  • A chave não está expirada ou desativada
