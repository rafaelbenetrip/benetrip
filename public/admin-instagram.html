Como configurar o Instagram

  1. Crie um app em developers.facebook.com
  2. Conecte sua conta Instagram Business a uma Facebook Page
  3. No Graph API Explorer, gere um token com permissoes: instagram_basic, instagram_content_publish, pages_show_list
  4. Troque por um long-lived token (valido por 60 dias)
  5. Obtenha o Instagram Business Account ID via Graph API
  6. Configure no Vercel:
    INSTAGRAM_ACCESS_TOKEN = seu token
    INSTAGRAM_BUSINESS_ID = ID da conta
  7. O cron roda automaticamente todo dia as 9:00 BRT (12:00 UTC)
